#  数组练习

##  合并两个有序数组——排序法

```c
class Solution {
public:
    void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) {
          
         for(int i = 0; i < n; i++)
         {
            nums1[m+i] = nums2[i];
         }
         sort(nums1.begin(), nums1.end());

                
    }
};
```

##  删除有序数组中的重复性——双指针法

left指针指向输出数组，right指针指向输入数组

```c
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        int left = 0;
        int right = 0;
        int n = nums.size();
        for(int i = 0; i < n; i++)
        {
            if(nums[right] != nums[left])
            {
                nums[left+1] = nums[right];
                left++;
                right++;
            }
            else
            {
                right++;
            }
        }
        return left + 1;
    }
};
```
